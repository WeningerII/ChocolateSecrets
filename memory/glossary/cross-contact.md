# Cross-Contact

**Definition.** Unintended transfer of an allergen from one food/surface to
another (distinct from cross-*contamination*, which is microbial). Safety-critical.

**In this codebase.** Modelled as risk data on ingredients/recipes (see
[[food-safety-allergens]]); evolved via `scripts/migrate-cross-contact-risks.mjs`.
