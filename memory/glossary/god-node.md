# God Node

**Definition.** In the Graphify code graph, a most-connected node — a core
abstraction many other symbols depend on. High-signal entry point for
understanding a codebase.

**In this codebase.** Listed in `../graphify-out/GRAPH_REPORT.md` → "God Nodes"
(e.g. `Ingredient`, `Recipe`, `useData`, `RecipePhysics`). Also in [[system-overview]].
