# Weighted-Average Cost (WAC)

**Definition.** Ingredient unit cost computed as a quantity-weighted average across
received [[lot|lots]] — smooths price changes between purchases.

**In this codebase.** Drives recipe costing in `src/services/culinaryTools.ts`;
lots come from [[inventory-and-lots]].

**Related.** [[lot]] · [[recipes-and-costing]] · [[inventory-and-lots]]
